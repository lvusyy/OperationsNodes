 
####  计算表达式
    expr 其中特殊符号需要转义 \* \. \> \<   变量 需要$
    $[表达式] 不需要转义 相等判断 需要 ==   可以不需要$
    $((和$[]))相同
    let i++ i-- --i ++i -= += *= /= 	//不需要转义 不需要$ 引用
    bc 支持小数计算
      echo "scale=3;3.14*2"|bc   //scale 只能限制除法时的精度
      
#### 条件判断
      [  表达式  ]
      test 表达式                      //和 [ 表达式 ] 都是一样的
      
      -z 空字符串 -n 非空字符串
      逻辑判断
      && and
      || or 或
      
      数值比较
      -eq -ne 
      -le -lt
      -ge -gt
      
    *bash中计算表达式 性能测试*
	  test -f file
	  [ -f file ]
	  [[ -f file ]] 执行效率最好, 前面两种性能差不多. [ ] 稍微比 test 好一点
